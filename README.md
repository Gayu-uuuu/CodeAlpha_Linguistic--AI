 Linguist AI
Linguist AI is a high-performance, intelligent translation platform that leverages the power of Google's Gemini AI to provide accurate, context-aware translations. Designed with a focus on speed and user experience, it offers a seamless way to communicate across 18+ languages.
✨ Key Features
🤖 AI-Driven Accuracy: Powered by the gemini-3-flash-preview model for superior linguistic context.
🔍 Auto-Detection: Instantly identifies the source language so you don't have to.
🔊 Text-to-Speech: Integrated accessibility to hear translations in their native accent.
📋 One-Tap Copy: Quickly copy translated text for use in other applications.
🎨 Modern UI/UX: A sleek, bento-style interface built with Tailwind CSS 4 and Motion.
📱 Fully Responsive: Optimized for desktop, tablet, and mobile devices.
🛠️ Tech Stack
Framework: React 19
Language: TypeScript
AI Engine: Google Gemini AI (@google/genai)
Styling: Tailwind CSS 4
Animations: Motion
Icons: Lucide React
Getting Started
Clone the repo: git clone https://github.com/your-username/linguist-ai.git
Install dependencies: npm install
Set up Environment: Add your GEMINI_API_KEY to a .env file.
Run Dev Server: npm run dev
